# groupie

- Features
  1) The application offers real-time messaging capabilities within groups, allowing users to engage in instantaneous conversations with other group members.
  2) Users can securely create accounts or log in using  email/password .
  3) Users have the ability to form their own groups, explore diverse group options, and participate in additional groups by joining them.
  4) The frontend is developed with Flutter and Dart, while Firebase is employed for backend services.
